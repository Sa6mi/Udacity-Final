

# MystoreFinal

MystoreFinal is a single-page application built with Angular. It presents a list of products that can be added to a cart and checked out. Key features include:
- View list of products
- Add products to cart
- View products in cart
- Delete products from cart
- Checkout products in cart
- Calculate cart total

## Getting Started

Clone the repository, install dependencies, and start the development server.

### Install Dependencies
```
npm install
```

### Start Server
```
ng serve
```

Open the app at:
```
http://localhost:4200/
```
